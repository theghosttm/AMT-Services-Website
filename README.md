🟣 AMT Services

AMT Services is a web-based Administrator Management Tool (AMT) designed for Gorilla Tag modding.
It acts as a Console GUI Website that can be connected to a compatible mod menu to unlock and manage administrator-level services directly inside your menu.

🌐 What Is AMT Services?

AMT Services provides a centralized web interface that allows authorized users to:

Connect their mod menu to an external service

Enable or disable administrator-only features

Manage permissions and advanced menu behavior

Securely link accounts (e.g., Discord integration)

Extend mod menu functionality without hardcoding everything

The website works as a bridge between your mod menu and external admin tools, making it easier to manage powerful features safely and efficiently.

🔧 Key Features

🧩 Mod Menu Integration
Designed to connect with Gorilla Tag mod menus that support external services.

🛡️ Administrator Services
Unlock special admin-only options such as moderation tools, enhanced controls, or developer utilities.

🖥️ Console-Style GUI
Clean, simple interface inspired by in-game console menus.

🔗 External Connections
Supports account linking (such as Discord) for authentication and role-based access.

⚙️ Expandable System
Built to allow future services, panels, and admin tools to be added easily.

🧠 How It Works (High-Level)

User visits the AMT Services website

User connects their account (ex: Discord or custom auth)

Mod menu communicates with AMT Services

Admin permissions are verified

Administrator features are enabled inside the menu

⚠️ Exact implementation depends on the mod menu and networking setup used.

🚀 Use Cases
